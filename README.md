# 🥯 Grumpy Bagels 

> A responsive, multi-page digital storefront and menu website designed to showcase a local bakery cafe.

## 🎯 The Problem
Local cafes and bakeries often rely on social media or PDF menus, lacking a dedicated, engaging digital presence. Customers need an intuitive way to explore the menu, learn about the brand's story, and find store locations without navigating through a cluttered interface.

## 💡 The Solution
This project is a custom-built, multi-page static website acting as a complete digital storefront for "Grumpy Bagels." Built entirely from scratch without the use of website builders, it demonstrates a strong understanding of web fundamentals, semantic markup, and cascading style sheets to create a warm, inviting user experience.

## ✨ Key Features

*   **Multi-Page Architecture:** Fully routed navigation across dedicated pages for the Home, About, Menu, Store Location, and Socials.
*   **Semantic HTML5 Structure:** Clean, accessible code utilizing proper `<header>`, `<nav>`, `<main>`, `<section>`, and `<footer>` tags.
*   **Centralized Styling (External CSS):** A globally linked `style.css` file ensures consistent branding, colors, and typography across all 5+ pages of the website.
*   **Rich Media Integration:** Seamlessly embeds optimized images and dynamic GIFs to visually showcase the baking process and menu items.
*   **Custom UI Elements:** Features a custom navigation bar and structured layout design that prioritizes readability and user flow.
